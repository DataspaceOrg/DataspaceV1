# Dataspace

Dataspace is a cloud integrated web application that helps streamline the process of managing and analyzing largee datasets. Allows you to upload, view, and analyze your datasets with AI querying. 

## Features

- Create a personal user for holding datasets
- Upload datasets to Cloud Storage
- Saved AI Based aggregations of data to give quick overview of information
- Build analytical models that help describe data.
- Ability to perform personal querying & retrieve additional insights from the data.
- Edit/Manage Datasets with GUI and Rest framework that allows to connect this to other applications. 

## Purpose

I wanted to work on this project as I often found myself having a bit of trouble when I would be trying to understand datasets for the first time, I also found managing/storing important datasets to be a bit of a difficult process, my main goal with this project is to have a simple and efficent way to manage and visualize datasets to retrieve information quickly and easily. It's also a project that helps challenge my skills and learn new technologies.
