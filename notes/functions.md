## db_services.py Helper functions

- detect_upload_type: Detects the type of the file being uploaded.
- save_raw_file: Saves the raw file (any type) to the dataset directory, under its uuid. 
- save_parquet_file: saves the parquet file to the dataset directory. under its uuid.

Parquet Data Type for ETL Transformation of data.
